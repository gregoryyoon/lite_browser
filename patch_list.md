# 개선사항
## 네이버에서 링크 클릭시 새창이 뜨는것 대신 새탭이 뜨도록 
## 프로그램 아이콘 디자인
## 마크다운 에디터창에 파일 관리하는 기능 개선
다음 기능을 위한 파일 관리 모듈과 그에 따른 GUI 생성
- Root 폴더 지정 기능
- Root 폴더내에서 서브폴더를  추가 기능
- Root 폴더내의 MD 파일들을 리스트로 보여주는 기능
- 리스트의 MD 파일들을 선택하여 불러오는 기능
- Root 폴더 혹은 서브폴더에서 MD 파일을 새로 생성하는 기능
